# GeekVanlife

Projet de documentation pour GeekVanlife, avec l'objectif de devenir une application à terme.

## Objectif

- Centraliser la documentation, guides, et ressources autour du projet Geek Valnife.
- Préparer une base solide pour une future application.

## Structure du projet

- `docs/` : Documentation, guides, tutoriels, schémas, etc.
- `src/` : (À venir) Code source de l'application.
